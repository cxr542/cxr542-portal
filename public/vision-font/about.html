Vision Font

시력에 맞춘 글자 크기로 웹 기사를 읽는 모바일 앱(MVP). React Native + Expo, Gemini API는 지식 베이스용 cloud-chatbot과 분리된 읽기 전용 클라이언트입니다.

흐름

① 시력 테스트 ② 맞춤 설정 ③ 기사 URL
  • 5단계 한글 시력표 → 폰트·줄간격·자간 자동 계산
  • localStorage에 설정 저장
  • WebView에 CSS injection (위키 등 일부 사이트) — 웹 데모는 샘플 기사에 적용
  • 웹 미리보기: 포털 /vision-font/
